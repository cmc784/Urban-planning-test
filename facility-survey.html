특정시설 운영현황 및 사례조사
산업단지뿐 아니라 개발제한구역·택지개발지구·정비구역 등 다양한 지역지구 내에서, 폐기물처리시설을 포함한 다양한 도시계획시설의 소재지·건축물 매칭 여부를 확인하는 범용 중첩분석 도구입니다.
아래 질문에 답하면 구역·시설 유형에 맞춰 업로드 항목과 판정 기준이 자동으로 구성됩니다. 분석 결과는 사례조사 기록으로 등록해 지속적으로 관리할 수 있습니다.
Q1. 어떤 구역(지역지구) 내에서 시설을 찾으시나요?
중첩분석의 기준이 될 구역 경계 SHP의 성격을 선택하세요.
Q1-1. 더 세분된 지역지구 선택이 필요한가요?
해당 없으면 "아니요"를 선택하고 바로 다음 질문으로 넘어가세요.
세분류를 모두 선택하세요 (복수 선택 가능)
Q2. 찾으시려는 시설은 무엇인가요?
먼저 시설을 어떤 기준으로 고르실지 선택하세요.

Q3. 운영여부 판정에 사용할 주용도 키워드를 확인·수정하세요
건축물통합정보의 주용도 필드에 아래 키워드 중 하나라도 포함되면 "운영 가능성 높음"으로 판정합니다. 쉼표로 구분해 자유롭게 수정하세요.
Q4. 공간정보 API 인증키가 있으신가요?
VWorld(국토교통부 공간정보 오픈플랫폼) 인증키가 있으면 자동조회를 기본으로 준비하고, 없으면 SHP 파일 직접 업로드를 기본으로 안내합니다. 어느 쪽을 고르든 이후 항목별로 언제든 전환할 수 있습니다.
인증키가 등록된 도메인에서 이 페이지를 열어야 정상 동작합니다. 요청이 실패하면 각 항목에서 파일 업로드로 개별 전환할 수 있습니다.
👍 각 항목에서 SHP(.zip) 파일을 직접 업로드하시면 됩니다. (필요해지면 이후에도 API 조회로 전환 가능)
Q5. 조회할 행정구역을 선택하세요 (선택사항, 여러 개 가능)
범위를 좁히지 않으려면 "전국"을 선택하세요. 시/도만 선택하고 시/군/구·읍/면/동은 생략해도 됩니다. 여기서 고른 행정구역은 이후 분석 결과를 해당 지역으로만 걸러내는 데 사용할 수 있습니다.

지도 영역 지정과 Q5 행정구역 선택은 둘 다 선택사항이며 같은 층위의 조회 범위 좁히기 수단입니다 — 아무 것도 지정하지 않아도 조회는 진행되며, 필요한 쪽만 골라 쓰면 됩니다.
VWorld 오픈API(vworld.kr)에서 인증키를 발급받고, 개발계정에 이 페이지가 열리는 도메인을 등록해야 정상 조회됩니다. 도메인이 다르면 브라우저 정책(CORS)·서버 검증에 의해 요청이 차단될 수 있으며, 이 경우 파일 업로드 방식을 이용해 주세요.
✅ ①구역·②시설 카드는 국토교통부 "용도지역지구 표준분류코드"에 근거해 Q1/Q2 답변에 따라 서비스ID가 자동으로 채워집니다. 직접 확인하고 싶다면 카드의 "🔍 VWorld 카탈로그 열기"로 이동해 대조해보세요.
⚠️ ③지적도(LP_PA_CBND_BUBUN)는 미리 확인된 값이 입력되어 있고, ④건축물 정보데이터는 이 표준분류코드 체계에 포함되지 않아(별도 data.go.kr WFS) 자동연결이 안 됩니다 — 파일 업로드를 이용해주세요.
